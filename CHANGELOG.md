# Changelog

All notable changes to this project will be documented in this file.

## 1.0.1 - 2026-03-12

- 移除贪吃蛇游戏功能及相关代码。
- 优化项目结构，专注于 Agent Plugin 智能体插件分发。
- 同步更新 README.md 与 package.json 资源清单。

## 1.0.0 - 2026-03-12

- Initial release of Queen Bee Swarm.
- Added 1 visible Queen Bee agent.
- Added 16 hidden worker agents.
- Added publish-ready project structure and maintenance documentation.
- Switched the primary distribution story to Agent Plugin packaging and marketplace installation.
- Added a dedicated plugin packaging script and aligned plugin metadata across manifests.
- Added a real VS Code chat participant entrypoint for `@queen-bee`.
- Improved the extension activation flow and cleaned up command/error handling.
- Upgraded the Snake webview to reuse an existing panel and use safer resource URIs.
- Updated the README to reflect Agent Plugin-first installation and VS Code compatibility guidance.
